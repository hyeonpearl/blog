## Commit rules

- Commit message: [형식] 글, 파일 이름 / 상세 설명

- 형식 예시
  - [POST] : 게시글 포스팅
  - [EDIT] : 파일 및 문서 수정
  - [CHORE] : 라이브러리 설치 등 기타 작업
  - [DELETE] : 파일 삭제
